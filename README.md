# SimulacionInteractivo-2026-20

## Introduccion

Este repositorio contiene las actividades, comentarios y entregas del curso Simulacion de sistemas interactivos 2026-20. Cada unidad del curso estara dividida en su propia carpeta para mantener el orden entre entregas y poder acceder a la informacion necesaria de manera mas directa.

Referencias, inspiraciones y otros recursos para el desarrollo de cada una de las actividades tendran su propia subcarpeta dentro de la carpeta propia de la unidad.
